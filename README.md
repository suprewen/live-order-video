# 直播点单视频模板

Remotion 驱动的直播点单素材生成器：**保留原视频**，叠加视频通话风格 UI，并在右上角预留纯绿商品位，供 OBS / 直播伴侣色度键叠加商品、菜单或订单页面。

## MVP 能做什么

- 将任意人物视频裁为 9:16 视频通话画面；
- 原样保留人物、动作、口型与声音；
- 固定底部通话控制 UI；
- 右上角固定 `#00FF00` 绿幕商品位；
- 可通过参数替换通话名称；
- Web 页面用于上传本地素材并实时预览；
- Remotion CLI 在本机渲染最终 MP4。

## 本地开发

```bash
npm install
npm run dev
```

访问终端显示的本地地址。网页端的文件上传仅用于预览，不会上传视频。

## 渲染 MP4

```bash
npm run render -- /绝对或相对路径/人物视频.mp4
```

可指定输出文件和通话名称：

```bash
npm run render -- ./input.mp4 ./out/order-call.mp4 "老张"
```

输出为 720×1280、30fps、H.264 MP4。渲染前会先将输入文件标准化为浏览器可渲染的 H.264，再交由 Remotion 合成；临时文件位于 `public/render-input.mp4`，已被 Git 忽略。

## 直播叠加商品

1. 把生成的 MP4 添加进 OBS 或直播伴侣。
2. 对该视频图层添加“色度键 / Chroma Key”。
3. 选取绿色 `#00FF00`。
4. 将商品图、菜单或点单网页图层置于视频图层下方。

## 部署说明

Cloudflare Pages 托管的是项目的预览与说明页面。最终 MP4 渲染需要 Remotion/FFmpeg，因此 MVP 在本地执行，不在 Pages 服务端渲染。
